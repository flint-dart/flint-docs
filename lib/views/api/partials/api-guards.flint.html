Auth Guards

Protect routes with middleware and request auth helpers.

Flint does not require a separate guard system. Use middleware and req.user to enforce access.

Example

class AuthGuard extends Middleware {
  @override
  Handler handle(Handler next) {
    return (req, res) async {
      final user = await req.user;
      if (user == null) {
        return res.status(401).json({'error': 'Unauthorized'});
      }
      return next(req, res);
    };
  }
}

app.get('/dashboard', (req, res) async => res.send('ok'))
  .useMiddleware(AuthGuard());
